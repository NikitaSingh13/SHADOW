git clone https://github.com/NikitaSingh13/Shadow.git
cd Shadow

create virtual env syntax -> conda create -p venv python==3.10 -y
activation syntax -> conda activate shadow_env  || agar ye activate kaam nah kare to -> activate -> conda activate C:\Users\hp\Desktop\Shadow\venv
                                                                    deactivate -> conda deactivate
dependencies install (only after the virtual env is activated ) syntax -> pip install -r requirements.txt

for openai-whisper and moviepy we need ffmpeg so syntax is -> python setup_ffmpeg.py
